ウクライナ戦争が今も続き、世界を席巻した新型コロナウイルスの流行は3年目に入った……<br>世の中はどうなっているんだ？この世界に生きる自分をどう理解すればいいのでしょうか？これらの問題を考える人が増えている。<br>現在に目を向けると、中国だけが依然としてグローバル化を積極的に擁護し、人類運命共同体の構築を推進しているようだ。<br><br>もしかしたら…バーチャルサークルの皆さんは、昔の好みや雰囲気の違いで溝ができていますが、雰囲気の違いは私たちが一緒に楽しみを求めることができるかどうかに影響しますか？<br>いいえ、みんなの心は実は善良で、みんなの初心は変わっていません。すべて「バーチャル」という存在のために集まっています。A-SOULでも、ホロライブでも、にじさんじでも、VirtuaRealでも、Overideaでも、他のプロジェクトや個人勢いであれ、みんなが調和して共存できると信じています。<br><br>もし誰かがこれらが実現不可能だと言ったら、私は…また傷跡を明らかにしますか？<br>ゲームは人が楽しむものだったはずなのに、いつのまにか憎しみを作る道具になってしまった。<br>後ろの話は多くの人に理解されにくいだろうが、このような無意識に城を囲むのはほどほどでいいよと思います。<br>原神、アークナイツであれ、幻塔、崩壊3rd、アズールレーン、Minecraftのプレイヤーであれ、実はみんなが仲良く一堂に楽しく集うことができます。<br>各方面の争いは今も残っているが、分岐の消失は時間の問題にすぎない。<br>もし誰かがこれらの願いは実現不可能だと言ったら、それは彼が歴史に淘汰される運命にあるものだけを見たことだ。<br><br>表面的なケンカしか見ていない人が多いが、その悪口に気が狂っている人もいることを知らない。<br>その狂者たちが獣のように隔たりの高い壁にぶつかったとき、多くの冷酷者が彼らを罵ったり笑ったりしていた。<br>しかし、その冷酷者たちはなぜ憎しみの製造者を非難しないのだろうか？<br>彼らは冷淡で非情で、彼らは強弱を頼みにして、憎しみの製造者を直視することができなくて、悪語に狂ってしまった人をいじめることを選んだ。<br><br>しかし、この状況は長く続くのだろうか？<br>できません。<br>矛盾がある程度で蓄積されると、その圧力は必ず爆発する。<br>恨み者や冷酷者たちは、狂者の口を塞いでいれば何も起こらないと思っている。彼らは洪水を塞いでいれば、洪水はこれ上昇しないと思っている。<br>その結果、より激しくなることの方が多い。<br>恨み者や冷酷者たちは、自分たちが多くの人を怒らせてはいけないと思ったのでしょう。<br><br>その隔たりの壁は昔の環境に適していたのかもしれませんが、もはや今は存在し続ける必要はありません。<br>その狂者たちは獣になり、隔たりの壁をぶち壊し、彼らが勝ち取った日の光を多くの人に感じてもらうために。<br>その狂者たちは苦痛に耐え、恨み者に挑戦したほうがましだ、この光彩陸離な世界をもっと美しくするためだけだ。<br><br>恨み者たちは1つのや2つの狂者に耐えられるが、彼らは何千もの狂者に抑えるこのだろうか？彼らは団結し一心にして、困難を恐れない狂者に抑えるこのだろうか？彼らは山川と大地を愛し、詩と遠方を愛し、希望と美しさを愛する狂者に抑えるこだろうか？<br>彼らはできない。<br>世界は複雑に入り組んでいて、恨み者は自分の無能さに気づけず、結局は自分が生み出した憎しみに振り回されることになるのです。<br>それと同時に、隔たりの壁が崩れているのを見て、狂者たちがこのことをする意味を見ました。<br>彼らは実際の行動で証明し、すべてが可能であることを証明した。<br>“みんなもそろそろお互いに交流する時間だ。”<br><br>偶然にも、狂者たちの中に歴史の影を見たのです。<br>1998年の長江洪水、2008年の中国雪害、四川地震、そして世界金融危機……それでも、これらの困難は苦難を経験した中国人民を打ち負かすことなく、世界を驚かせた北京オリンピックを成功させた。<br>“われわれ中華民族は強い凝集力を持っており、いかなる困難もわれわれを圧倒することはできない、中国人民は勝つことはできない！”<br>変革が起こり、希望が到来し、みんなも団結し始め、彼らを負かすことができないものは、彼らをさらに強くするに違いない。<br><br>もし誰かが新科娘がいないと言ったら、それは彼らが現実に直視したくないだけだ。<br>彼らの目には、新科娘は卒業したVTuberにすぎず、今後で彼女を知る人はいないだろう。<br>しかし心萪は、バーチャルサークルだけでなく、二次元にも
